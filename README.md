## 电信线路：从 163 到 CN2 GIA 的四个档次

电信用户最多，线路体系也最复杂，分四档。

**第一档：163 骨干网（AS4134）**

最基础，最便宜。几乎所有"入门级"VPS 走的都是这条。优点是覆盖广，缺点是晚上 8 点到 11 点容易炸——丢包率有时能到 30%，看个视频都能感觉到卡顿。

**第二档：CN2 GT（AS4809，半程 CN2）**

**第三档：CN2 GIA（AS4809，全程 CN2）**

全程走 CN2 专网，进国内之后不回落 163，一路都是 `59.43.x.x`。这是电信最高档的国际线路，晚高峰基本不受影响，延迟稳，丢包极少。
